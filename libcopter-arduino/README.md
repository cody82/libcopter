# libcopter for ESP32

# Connections

|  ESP32 Pin | Function | Input |
|---|---|---|---|---|
| GPIO36 | ADC1_CH0  | Throttle |
| GPIO39 | ADC1_CH3 | Pitch |
| GPIO34 | ADC1_CH6 | Yaw |
| GPIO35 | ADC1_CH7 | Roll |
| GPIO25 | Digital Input | Takeoff |
| GPIO26 | Digital Input | Land |
| GPIO27 | Digital Input | Panic |
| GPIO14 | Digital Input | Calibrate |
